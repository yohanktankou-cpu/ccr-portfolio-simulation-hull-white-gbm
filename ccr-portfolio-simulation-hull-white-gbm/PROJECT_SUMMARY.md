# Project Summary for Recruiters

This project is a clean public version of a counterparty credit risk and portfolio valuation workflow.

## What the project does

- Simulates correlated EUR and USD interest-rate paths and EUR/USD FX paths.
- Uses Hull-White one-factor style dynamics for rates and GBM for FX.
- Values a synthetic two-swap portfolio under the simulated scenarios.
- Converts USD MtM into EUR using the simulated FX path.
- Aggregates trades at portfolio/netting-set level.
- Applies simplified collateral logic.
- Computes Expected Exposure and 95% Potential Future Exposure.

## Why it matters

The project demonstrates the link between stochastic modelling, derivatives valuation, portfolio aggregation and risk metrics used in counterparty credit risk and quantitative risk modelling.
